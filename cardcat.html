  2. Confirm the title is not already present in the Academy/Drexel Library Catalog. If the title is already present in the catalog, please start over with a new card.
  3. Search for the title in WorldCat.
  4. If you find an exact match, complete the form by copying and pasting relevent information from WorldCat. If you cannot find an exact match, please start over with a new card.
  5. Complete as much information as you can in the form on the right, and hit Submit. Then you can start again with a new card.

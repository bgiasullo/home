Catalogging instructions
  1. Make sure you can confidently read the text on the card. If you cannot read the text, please start over with a new card.
  2. If you have already seen the displayed card, start over with a new card
  3. Confirm the title is not already present in the Academy/Drexel Library Catalog. If the title is already present in the catalog, please start over with a new card.
  4. Search for the title in WorldCat.
  5. If you find an exact match, complete the Card Catalog Form by copying and pasting relevent information from WorldCat and editing minor details as needed. If you cannot find an exact match, please start over with a new card.
  6. Complete as much information as you can in the Card Catalog Form on the right, and hit Submit. Then you can start again with a new card.
